ASP-Based Epistemic Collaborative Planning for Anonymous Task-Sequential Multiagent Path Finding

Hamit Efe Çınar, Pere Pardo Ventura

Manuscript in preparation

In Preparation

This work addresses a variant of the Multi-Agent Path Finding problem featuring partial observability, decentralization, anonymous task-sequential requirements with constrained communication. The framework models observation and communication using dynamic epistemic logic and employs Answer Set Programming for collaborative planning under uncertainty. The approach includes simulation capabilities to compare performance across different exploration methods and agent configurations.
