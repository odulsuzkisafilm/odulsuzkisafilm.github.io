Partial Observable Multi-Agent Path Finding with Constrained Communication

Erasmus Internship, University of Luxembourg

July - Sep 2025

Supervisors: Prof. Leon van der Torre & Dr. Pere Pardo Ventura
Focus: Multi-Agent Systems, Epistemic Planning, Answer Set Programming

Designed a framework to solve a novel variant of the Multi-Agent Path Finding (MAPF) problem featuring partial observability, decentralization, and anonymous task-sequential requirements with constrained communication. Modeled observation and communication using dynamic epistemic logic and employed Answer Set Programming for collaborative planning under uncertainty. Developed a simulation script to compare performance across different discovery methods and agent configurations. Currently preparing a manuscript for publication.
